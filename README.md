# Expense Tracker API

A simple ASP.NET Core Web API for managing expenses.

## Features

* Get all expenses
* Get expense by ID
* Add expense
* Update expense
* Delete expense

## Technologies Used

* ASP.NET Core Web API
* C#
* Git & GitHub

## Author

Fizza Azhar

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio.
3. Run the project using `dotnet run` or press F5.
4. Open the Swagger page in your browser.
5. Test the API endpoints (GET, POST, PUT, DELETE).
